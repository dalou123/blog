    CSS Grid布局

    dalou 2023/5/20 css3

    # grid布局是什么

    Grid布局即网格布局，是一个二维的布局方式，由纵横相交的两组网格线形成的框架性布局结构，能够同时处理行与列。擅长将一个页面划分为几个主要区域，以及定义这些区域的大小、位置、层次等关系， 与flex的一维布局并不相同。

    设置了display:grid/inline-grid的元素就是网格布局容器，这样就能触发浏览器渲染引擎的网格布局算法。

    <div class="container">
    <div class="item item-1">
        <p class="sub-item"></p>
    </div>
    <div class="item item-2"></div>
    <div class="item item-3"></div>
</div>

    # 属性

    Grid属性可分成两大类：

    • 容器属性
    • 项目属性

    # display属性

    在元素上设置display:grid或display:inline-grid来创建一个网格容器

    • display:grid：该容器是一个块级元素
    • display:inline-grid：该容器元素为行内块元素

    # grid-template-columns/rows属性

    • grid-template-columns：设置列宽
    • grid-template-rows：设置行高
    .container{
    /* 设置容器的显示方式为网格布局 */
    display: grid;
    /* 定义网格的列宽，这里有三列，每列宽度为200px */
    grid-template-columns: 200px 200px 200px;
    grid-gap:5px;
    /* 定义网格的行高，这里有两行，每行高度为50px */
    grid-template-rows: 50px 50px;
}

    以上表示固定列宽为200px 200px 200px，行高为50px 50px。可以看到，在上面代码中重复写了单元格的宽高，通过使用repeat（）函数，可以简写重复的值。

    • 第一个参数是重复的次数
    • 第二个参数是重复的值 所以以上代码可简写为：
    .container{
    display: grid;
    grid-template-columns: repeat(3,200px);
    grid-gap:5px;
    grid-template-rows:  repeat(2,50px);
}

    • auto-fill:表示自动填充，让一行（或者一列）中尽可能的容纳更多的单元格

    grid-template-columns：repeat(auto-fill, 200px)表示的是列宽200px,但列的数量不固定，只要浏览器容纳得下，就可以放置元素

    • fr: 片段，为了方便表示比例关系

    grid-template-columns：200px 1fr 2fr表示第一个列宽设置为200px，后面剩余的宽度分为两部分，宽度分别为剩余宽度的1/3,2/3

    • minmax:产生一个长度范围，表示长度在这个范围之中都可以应用到网格项目中。第一个参数为最小值，第二个参数为最大值

    minmax(100px, 1fr)表示列宽不小于100px，不大于1fr

    • auto：由浏览器自己决定长度

    grid-template-columns: 200px auto 200px表示第一第三列为100px，之间由浏览器决定长度

    # grid-gap属性

    grid-row-gap属性、grid-column-gap属性分别设置行间距和列间距，grid-gap是两者的简写

    • grid-row-gap：10px表示行间距是10px
    • grid-column-gap： 20px表示列间距是20px
    • grid-gap：10px 20px等同于上述两个属性

    # grid-template-areas属性

    用于定义区域，一个区域由一个或者多个单元格组成

    .container{
    display: grid;
    grid-template-columns: 100px 100px 100px;
    grid-template-rows: 100px 100px 100px;
    grid-template-areas: 'a b c'
                         'd e f'
                         'g h i';
}

    # grid-auto-flow属性

    划分网格以后，容器的的子元素会按照顺序，自动放置在每一个网格，顺序就是由grid-auto-flow决定，默认为行，代表先填满第一行，再开始放入第二行

    # justify/align-items/self属性

    • justify-items属性设置单元格内容的水平位置（左中右
    • align-items属性设置单元格的垂直位置（上中下）
    • justify-self：设置单元格内容的水平位置（左中右），与justify-items用法完全一致，但只作用单个项目
    • align-self:设置单元格内容的垂直位置（上中下），与align-items用法完全一致，但只作用单个项目
    .container{
    justify-item: star | end | center | stretch;
    align-items:  star | end | center | stretch;
}

.item{
    justify-self: star | end | center | stretch;
    align-self:  star | end | center | stretch;
}

    属性对应如下：

    • start：对齐单元格的起始边缘
    • end：对齐单元格的结束边缘
    • center：单元格内部居中
    • stretch：拉伸，占满单元格的整个宽度（默认值） place-items属性是align-items属性和justify-items属性的合并简写形式,place-self属性是align-self属性和justify-self属性的合并简写形式.

    # justify/align-content属性

    • justify-content属性是整个内容区域在容器里面的水平位置（左中右）
    • align-content属性是整个内容区域的垂直位置（上中下）
    .container{
    justify-content: star | end | center | stretch | space-around | space-between | space-evenly;
    align-content:  star | end | center | stretch | space-around | space-between | space-evenly;
}
    1
    2
    3
    4

    • start:对齐容器的起始边框

    • end：对齐容器的结束边框

    • center：容器内部居中

    • justify-content:space-around：每个项目两侧的间隔相等。所以，项目之间的间隔比项目与容器边框的间隔大一倍

    • justify-content:space-between：项目与项目的间隔相等，项目与容器边框之间没有间隔

    • justify-content:space-evenly：项目与项目的间隔相等，项目与容器边框之间也是同样长度的间隔

    # grid-column/row-xx属性

    • grid-column-start：左边框所在的垂直网格线
    • grid-column-end：右边框所在的垂直网格线
    • grid-row-start：上边框所在的水平网格线
    • grid-row-end：下边框所在的水平网格线

    通过指定网格项目所在的四个边框，分别定位在哪根网格线，从而指定项目的位置

    <style>
        .container{
            display: grid;
            grid-template-columns: 100px 100px 100px;
            grid-template-rows: 100px 100px 100px;
            grid-gap: 2px;
        }
        .item{
            background-color: antiquewhite;
        }
        .item-1{
            grid-column-start: 2;
            grid-column-end: 4;
        }
</style>

    <div class="container">
        <div class="item item-1">1</div>
        <div class="item item-2">2</div>
        <div class="item item-3">3</div>
    </div>
